## 49.用Python实现一个二分查找的函数。
```python
li = [i for i in range(10)]
li.sort()
def bin_select(li,value,min=0,max=False):
    if max is False:
        max = len(li)-1
    mid = (max-min)//2
    if max < min:
        return None
    elif value > li[mid+min]:
        min = mid+1+min
    elif value < li[mid+min]:
        max = mid-1
    elif value == li[mid+min]:
        return mid+min
    return bin_select(li,value,min,max)
```
